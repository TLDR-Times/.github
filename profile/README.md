# TLDR Times

AI-powered news summaries that cross language barriers.

TLDR Times collects news articles published in languages you don't read, then translates, summarizes, and categorizes them into the language you prefer — so you can stay informed without the friction.

## What we build

- **Multilingual summaries** — every article is available in every supported language.
- **Lightweight reading** — short summaries with a link back to the original source.
- **Anonymous community** — likes and threaded comments without an account.

## Stack

Spring Boot backend, modern web frontend, scheduled ingestion pipeline.

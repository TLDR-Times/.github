# 1. TLDR-Times

GitHub Organization for the **TLDR Times** service: AI-translated, AI-summarized foreign news delivered in the reader's preferred language.

## 1.1. Product

| Aspect | Value |
|---|---|
| What | Reads foreign news articles, produces short summaries, serves them translated. |
| Who for | Readers who want news written in languages they do not read. |
| MVP category | Foreign football articles only. |
| MVP languages | Korean, English. |
| Interaction | Anonymous likes and threaded comments per article. |

## 1.2. Repositories

| Repo | Role |
|---|---|
| `tldrtimes.org` | Primary product monorepo: backend (ingestion, translation, summarization, API) and frontend (article list and detail pages). |
| `.github` | Organization-wide defaults: PR template, profile, community health files. |

# .github

Organization-wide GitHub configuration for [TLDR Times](https://github.com/TLDR-Times).

This repository hosts shared community health files and the organization profile. Contents here apply as defaults to every public repository in the org that doesn't define its own equivalent.

## Contents

- `profile/README.md` — public landing page shown at https://github.com/TLDR-Times.
- `pull_request_template.md` — default PR template applied across org repositories.
